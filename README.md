Django Blog Project
This is a functional Blog application built using the Django web framework.

Features:
Post Display: Shows a list of blog posts on the homepage.

Backend Logic: Managed using Django's Model-View-Template (MVT) architecture.

Database: Uses SQLite for data storage.

Technologies Used:
Python

Django

HTML/CSS

I built this project to demonstrate my understanding of Django's core concepts like routing, views, and template rendering.
